# Loan Sanction Prediction

A supervised machine learning project that uses a Support Vector Machine classifier to predict loan approval outcomes from applicant and financial features.

## Workflow

- Inspect and prepare the dataset
- Encode and scale relevant features
- Train an SVM classification model
- Evaluate predictions
- Predict an approval outcome for new input data

## Technology

Python · pandas · NumPy · scikit-learn · Jupyter Notebook

## Usage

Open the notebook in Jupyter and run the cells in order to reproduce preprocessing, training, evaluation, and prediction.
